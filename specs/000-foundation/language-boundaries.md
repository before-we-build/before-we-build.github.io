# Public language boundaries

Status: public-safe wording rule.

The public Before We Build site should sound like a simple Christian preparation tool, not like a system that classifies people.

## Preferred public wording

Use:

- Scripture first;
- before God and Scripture;
- check the foundation;
- count the cost;
- wisdom, understanding, knowledge;
- plans and counsel;
- prayer, conscience, church, wise counsel;
- questions for reflection;
- private summary in the browser.

## Avoid on the public path

Do not lead with words that make the site feel like a people-sorting product:

- labels for people;
- fixed human categories;
- numerical ratings;
- fit formulas;
- hidden verdicts;
- promises of outcomes;
- fate language;
- claims that a tool can reveal God's will.

## Tone

Concrete, biblical, limited, and humble. Start from a passage, then draw only the questions that can responsibly follow from that passage.
