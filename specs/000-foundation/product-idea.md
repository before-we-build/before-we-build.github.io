# Product idea — public Scripture-first doorway

Status: public-safe foundation note.

Before We Build is now presented publicly as a Scripture-first preparation path before serious shared decisions.

The public site does not begin with a model of people. It begins with the question: what does Scripture call us to examine before we build?

## Public path

1. Read or choose a specific Scripture passage.
2. Answer limited reflection questions that flow from that passage.
3. Keep answers private in the browser unless the user deliberately exports them.
4. Bring unresolved questions to prayer, Scripture, conscience, church, and wise counsel.

## What the public site is not

- not a verdict about a person;
- not a hidden decision engine;
- not a promise about relationships, marriage, work, or ministry;
- not a replacement for Scripture, prayer, church, pastoral care, or safety help.

## Current MVP focus

The first public MVP is Start Alone / Personal Preparation Map: a static browser-only reflection path with five Scripture passages and passage-specific questions.
