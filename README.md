# Before We Build — калькулятор відношень

Статичний сайт GitHub Pages з одним дослідницьким інструментом: калькулятором типових відношень у соціоніці, психософії та темпористиці.

## Перевірка

```bash
node --test tests/*.test.mjs
```

## Локальний перегляд

```bash
python3 -m http.server 8080
```

Відкрийте <http://localhost:8080>.
