# Before We Build — сайт GitHub Pages

Статичний публічний сайт GitHub Pages для Scripture-first входу Before We Build.

Адреса сайту: <https://before-we-build.github.io/>

## Поточна публічна архітектура

Публічний сайт більше не веде до старої гілки опитувальників, підбору або класифікації людей.

Основний шлях:

1. `index.html` — простий вхід: Писання спочатку.
2. `foundation.html` — що означає перевірити оснування перед будівництвом.
3. `start-alone.html` — приватний локальний шлях: вибір місця Писання, питання, підсумок у браузері.
4. `christ.html` — короткий християнський відповідь про Христа і Євангеліє.
5. `limitations.html` — межі: сайт не дає вироків, не визначає Божу волю і не замінює живу допомогу.

## Локальний перегляд

```bash
python3 -m http.server 8080
```

Відкрийте <http://localhost:8080>.

## Примітки

- Статичний HTML/CSS/JS без зовнішніх залежностей.
- Відповіді в Start Alone зберігаються тільки в браузері користувача.
- Немає серверної відправки відповідей і немає автоматичного поширення результатів.
